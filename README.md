# Kansas Toxic Chemicals Exposure Disparity

This project visualizes disparities in exposure to toxic chemicals across Kansas. It uses census tract data combined with the EPA Toxic Release Inventory to calculate:

- **Black / White TB Ratio**
- **Black TB / Total TB**
- **White TB / Total TB**
- **Poverty TB / Total TB**

Interactive maps allow users to hover over tracts to see detailed population and toxic burden metrics.
